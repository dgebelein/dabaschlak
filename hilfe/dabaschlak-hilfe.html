Dabaschlak Hilfestellung

Was ist Dabaschlak?

Dabaschlak ist ein Dokumentationssystem für Arbeiten an Versuchen und Versuchsflächen. Es speichert verrichtete Versuchsarbeiten in einer Datenbank und generiert daraus Protokolle.
Eine integrierte Benachrichtigungsfunktion informiert den interessierten Wissenschaftler über die Arbeiten an seinen Versuchen und den Gärtner oder TA über neue oder aktualisierte Versuchspläne.

Workflow

1. Versuch bei der Datenbank registrieren

Ein geplanter Versuch muss immer zunächst bei der Datenbank registriert werden. Hierfür ist ein Formular mit einigen grundlegenden Daten auszufüllen:

  1. Klicken Sie auf der Linken Seite des Programmfensters auf 'Versuche - Liste aller Versuche'

    Rechts erscheint eine Liste der bereits registrierten Versuche für das aktuelle Jahr. Sie können diese zeitliche Einschränkung über die Auswahloption links unter der Liste ändern oder aufheben.

  2. Für einen neuen Versuch klicken Sie rechts unten auf die Schaltfläche 'Versuch hinzufügen'.

    Hinweise zum Ausfüllen des Formulars:

    • Ist ein Versuch auf mehrere Kabinen, Klimakammern, usw. verteilt, können bei der Standortauswahl auch mehrere Optionen angekreuzt werden.
    • Ganz wichtig: Der Versuchsplan wird über Dateiname und Speicherort in der Datenbank verknüpft, diese Eigenschaften dürfen sich nach der Registrierung nicht mehr ändern. (Selbstverständlich kann der Inhalt des Versuchsplans trotzdem jederzeit überarbeitet und ergänzt werden - wenn nur der Dateiname gleich bleibt.)
  3. Die Basisdaten eines bereits registrierten Versuchs ändern Sie, indem Sie eine Zeile markieren und im Kontextmenü (linke Maustaste) 'Basisdaten bearbeiten/auswählen' anklicken.

    Diese Daten kann sich grundsätzlich jeden ansehen. Ändern kann sie jedoch nur der für den Versuch zuständige Wissenschaftler. (also derjenige, der im Formular als 'Versuchsleiter' eingetragen ist.)

2. Arbeiten dokumentieren

Dabaschlak unterscheidet zwischen Arbeiten, die eindeutig einem Versuch zuzuordnen und Arbeiten, die - unabhängig von Versuchen - flächengebunden sind. (z. B. Bodenbearbeitung oder Grunddüngung im Herbst/Winter)

  1. Klicken Sie auf der Linken Seite des Programmfensters auf 'Arbeiten dokumentieren - an Versuchen' oder 'Arbeiten dokumentieren - an Flächen'

    In der Mitte erscheint - je nach Auswahl - eine Liste mit den aktuellen Versuchen oder den Versuchsflächen. Der dritte Bildschirmbereich ganz rechts ist zu diesem Zeitpunkt noch leer.

  2. Klicken Sie jetzt in der Liste einen Versuch oder eine Versuchsfläche an.

    Rechts werden die bereits ausgeführten Arbeiten für den Versuch/Standort angezeigt.

  3. Für eine neue Arbeit klicken Sie rechts unten auf die Schaltfläche 'neue Aktion'.

    Hinweise zum Ausfüllen des Formulars:

    • Je nach ausgewählter Aktion können weitere Informationen nötig sein. Ergänzen Sie diese im Feld 'Notizen' als formlosen Text.
    • Eine neue Arbeit kann erst dann in die Datenbank übernommen werden, wenn das Formular vollständig und korrekt (keine roten Rahmen mehr) ausgefüllt ist.
  4. Eine bereits vorhandene Arbeit ändern Sie, indem Sie eine Zeile markieren und im Kontextmenü (linke Maustaste) 'Aktion anzeigen/bearbeiten' anklicken.
3. Benachrichtigungssystem nutzen

Das Benachrichtigungssystem informiert über Arbeiten an ausgewählten Versuchen und über neue oder aktualisierte Versuchspläne.

  1. Wählen Sie 'Benachrichtigungssystem - Nachrichtenzuordnung' und tragen Sie sich dort in der Spalte 'Nachricht' als Interessent für ausgewählte Versuche ein.
  2. Danach können Sie mit 'Benachrichtigungssystem - Meine Nachrichten' eine Liste der relevanten Versuchsarbeiten abrufen.
  3. Eine nach Aktualisierungszeitpunkt sortierte Liste alle registrierten Versuchspläne erzeugen Sie mit 'Benachrichtigungssystem - Neueste Versuchspläne'

    Sie öffnen einen Versuchsplan, indem Sie eine Zeile anklicken und im Kontextmenü (rechte Maustaste) 'Versuchsplan öffnen' auswählen.

4. Protokolle erstellen
  1. Klicken Sie auf 'Berichte - Versuchsprotokoll erstellen'

    Im mittleren Abschnitt des Programmbereichs erscheint eine Liste der registrierten Versuche für das aktuelle Jahr. Sie können diese zeitliche Einschränkung über die Auswahloption links unter der Liste ändern oder aufheben.

  2. Durch einen Klick auf einen Versuch in der Liste wird ganz rechts das Protokoll der entsprechenden Versuchsarbeiten erzeugt.

    Sie können dieses Protokoll drucken, speichern oder direkt zu Excel übertragen.('Speichern' erzeugt ein Dokument im html-Format, das sich auch in Word einlesen und weiterbearbeiten lässt.)

5. individuelle Datenbankabfragen

Diese Funktion ist noch nicht implementiert.

6. allgemeine Hinweise zur Bedienung des Programms
  • Dabaschlak übernimmt den Benutzer vom System, also den Namen mit dem Sie sich beim Betriebssystem angemeldet haben. Ein Umschalten auf eine andere Person ist innerhalb des Programms nicht möglich.
  • Der Arbeitsbereich (rechter Teil des Programmfensters) kann zweigeteilt sein. Sie ändern das Größenverhältnis dieser Bestandteile, indem Sie die weiße Trennlinie dazwischen mit gedrückter Maustaste verschieben
  • Listen haben meistens ein Kontextmenü, das Sie aufrufen, indem Sie einen Listeneintrag mit der rechten Maustaste anklicken. Über dieses Menü können Sie weitere Aktionen starten (z.B. Versuchspläne öffnen).
  • Listen können umsortiert werden, indem Sie auf einen Spaltenkopf klicken. Ein weiterer Mausklick auf denselben Spaltenkopf kehrt die Sortierreihenfolge um
  • Die Spaltenbreite von Listen ändern Sie, indem Sie im Spaltenkopf die Trennlinie mit gedrückter Maustaste verschieben
  • Eingabeformulare können erst dann in die Datenbank übernommen werden, wenn keines der Eingabefelder mehr rot umrahmt ist. Dabaschlak liefert ggf. Hinweise zur Fehlerursache - bewegen Sie zu diesem Zweck den Mauszeiger über ein rotes Eingabefeld.
